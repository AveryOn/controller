## Controller (Desktop App)

### Description
Personal desktop application focused on application architecture and local data security.

### Scope
- Modular desktop application architecture
- Encrypted local storage
- Hierarchical data model (directories, files)
- Markdown-based structured content editor
- Custom database manager and migration system
- Session and token handling
- IPC-based communication between Electron processes

### Tech
- Electron
- SQLite
- SQLCipher
- Encryption

### Notes
- Architecture-focused project
- Emphasis on long-term maintainability and architectural trade-offs
